# ConnectFour
An AI enabled  ConnectFour game
A ConnectFour game is a simple game where one has to connect 4 pieces in any one of 3 ways - Horizontal ,vertical ,diagonal 

For GUI -pygames is used
For AI - MinMAX algo is used
